# iAforo

## Pre-requirements - Software

You need to install:
- **[Git](https://git-scm.com/downloads)** (Include Git Bash)
- **[Python](https://www.python.org/)**
- **[OpenCV](https://opencv.org/)**

Knowledge of: **Python**, **Numpy** and **OpenCV**.
